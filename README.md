# Sagittarius A* Relativistic Observatory

A self-contained, real-time WebGL2 visualization of Sagittarius A*, the supermassive black hole at the centre of the Milky Way.

The site uses Three.js from a CDN and performs Kerr-inspired backward ray tracing, procedural volumetric plasma emission, relativistic Doppler and redshift effects, hotspot flares, polarization overlays, and cinematic, EHT 230 GHz, and GRAVITY near-infrared observing modes.

Open `index.html` directly in a modern browser or visit the GitHub Pages deployment.
