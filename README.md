# Sagittarius A* Relativistic Observatory

A self-contained, real-time WebGL2 observatory with two selectable black-hole regimes: Sagittarius A* and a notional 100-million-solar-mass near-Eddington supermassive black hole.

The site uses Three.js from a CDN and performs Kerr-inspired backward ray tracing, procedural plasma emission, relativistic Doppler and redshift effects, hotspot flares, polarization overlays, and cinematic, EHT 230 GHz, and GRAVITY near-infrared observing modes. Sagittarius A* uses a geometrically thick, optically thin RIAF; the luminous target uses an optically thick thin/slim disk with NASA/Interstellar-style lensing geometry.

Open `index.html` directly in a modern browser or visit the [GitHub Pages deployment](https://tothur.github.io/sagittarius-a-observatory/).
